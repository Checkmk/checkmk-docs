== New function

In {CMK} versions prior to 2.1.0 unecrypted communication with hosts is the default.
{CMK} 2.1.0 introduces the _Agent Controller_ for encrypted communication and thus needs differentiating both modes.
Unencrypted communication ist fully covered in the article xref:agent_linux#[Monitoring Linux].
