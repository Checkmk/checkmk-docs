= Monitor a web server
:revdate: 2024-07-08
:title: Monitor a web server
:description: Checkmk provides active checks that allow you to monitor network services directly and conveniently from the outside, and to display the information in your monitoring.

[#create_a_folder]
== Create a folder

In the first step you need to create a folder.

[.text-lg]
Go link:/pass:[$$SITE_NAME$$]/check_mk/index.py?start_url=%pass:[$$SITE_NAME$$]%2Fcheck_mk%2Fwato.py%3Ffolder%3D%26mode%3Dfolder[here] to add a folder

image::onboarding_add_folder.png[alt="Illustration of a folder."]

++++1. Click on "*_Add folder_*"

image::onboarding_web_server_folder.png[alt="Illustration of 'Web Server' folder."]

++++2. Set "_Title_" to "*_Web Server_*"

++++3. Set "_Checkmk agent / API integrations_" to "*_No API integrations, no Checkmk agent_*"

++++4. Set "_Labels_" to "*_webprotocol:https_*"

++++5. Click on "*_Save_*" button

[#set_up_regular_active_check]
== Set up regular active check

[.text-lg]
Go link:/pass:[$$SITE_NAME$$]/check_mk/index.py?start_url=%pass:[$$SITE_NAME$$]%2Fcheck_mk%2Fwato.py%3Fgroup%3Dactivechecks%26mode%3Drulesets[here] to set up an active check

image::onboarding_check_http_web_service.png[alt="Illustration of active checks overview"]

++++1. Click on "*_Check HTTP web service_*"

++++2. Click on "*_Add rule_*"

image::onboarding_add_new_endpoint.png[alt="Illustration of 'Add endpoint button'."]

++++3. Click on "*_Add new endpoint_*"

image::onboarding_http_name_and_URL.png[alt="Illustration of 'Name' and 'URL' settings."]

++++4. Set "_Name_" to "*_Basic webserver health_*"

++++5. Set "_URL_" to "*_https://$HOSTNAME$/_*"

image::onboarding_http_check_thresholds.png[alt="Illustration of threshold settings."]

++++6. In "_Response time_" set "_Warning_" to "*_1s_*"

++++7. In "_Response time_" set "_Critical_" to "*_2s_*"

++++8. Set "_Status code_" to "_*200*_"

++++9. In "_Certificate validity_" set "_Warning below_" to "*_15 days_*"

++++10. In "_Certificate validity_" set "_Critical below_" to "*_8 days_*"

image::onboarding_http_host_conditions.png[alt="Illustration of host selection."]

++++11. In "_Host labels_" set "_Label is_" to "*_webprotocol:https_*"

++++12. Click on "*_Save_*"

[#activate_changes]

[.text-lg]
Go link:/pass:[$$SITE_NAME$$]/check_mk/index.py?start_url=%2Fpass:[$$SITE_NAME$$]%2Fcheck_mk%2Fwato.py%3Fmode%3Dchangelog[here] to activate changes

image::onboarding_activate_changes.png[alt="Illustration of activate changes."]

++++1. Click on "*_Activate on selected sites_*"

[#add_web_server_as_host]
== Add web server as host

[.text-lg]
Go link:/pass:[$$SITE_NAME$$]/check_mk/index.py?start_url=%2Fpass:[$$SITE_NAME$$]%2Fcheck_mk%2Fwato.py%3Ffolder%3Dwebsite%26mode%3Dfolder[here] to add your web server as a host

image::onboarding_add_host.png[alt="Illustration of how to add a host"]

++++1. Click on "*_Add host_*"

image::onboarding_web_host.png[alt="Illustration of how to configure host"]

++++2. Set "_Host name_" to the URL of the website your web server provides, e.g. "checkmk.com"

++++3. Click on "*_Save & run service discovery_*"

image::onboarding_activate_changes.png[alt="Illustration of activate changes."]

[#view_monitored_hosts]
== View your monitored hosts

++++1. Click on "*_Monitor_*" in the menu and enter "*_Basic webserver health_*" in the search and press enter

image::onboarding_web_server_service_search.png[alt="Illustration of the list of hosts with 'Basic webserver health' service."]

++++2. You now see all hosts with the "*_Basic webserver health_*" service you just configured
