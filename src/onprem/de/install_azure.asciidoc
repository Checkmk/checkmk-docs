[IMPORTANT]
====
Images mit vorinstalliertem {CMK} {v23} waren nie verfügbar und werden nie verfügbar sein.
Falls Sie ein Image mit {v22} gebucht haben, dann beachten Sie bitte die Hinweise zum 
xref:update#[Hinweise zum Update], die  xref:update_major#[detaillierten Informationen zum Update auf {v23}] und gegebenenfalls die xref:release_upgrade#[Hinweise zum Linux-Upgrade auf dem Checkmk-Server].
Falls Sie ein Image neu buchen wollen, wählen Sie bitte die Version dieses Artikels für die neueste stabile {CMK}-Version aus.
