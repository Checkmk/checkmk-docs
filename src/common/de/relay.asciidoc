== Introduction

ifdef::onprem[]
{cce-only}
endif::[]
Es gibt Fälle, in denen eine {CMK} Instanz zu überwachende Hosts nicht erreichen kann, beispielsweise wenn eine Firewall eingehenden Verkehr für ein Netzwerk blockiert.
Mit
ifdef::onprem[]
{UE}
endif::[]
ifdef::saas[]
{CE}
endif::[]
kann das {CMK} Relay innerhalb solcher Netze dazu verwendet werden, Monitoring-Daten einzusammeln und an die {CMK}-Instanz zu schicken.

Eine vollständige Beschreibung des gegenwärtigen Funktionsumfangs finden Sie bis auf weiteres in der link:../en/relay.html[englischen Version dieses Artikels].
